The suite's three type voices: `--suite-font-display` (Space Grotesk) for letter-spaced tool titles, `--suite-font-ui` (Inter) for UI and body, `--suite-font-mono` (JetBrains Mono) for code, prompt templates, and data readouts. UI text runs small (12–13px) and relies on the text-hierarchy tokens for emphasis.
