Horizontal row of `FormField`s — each field flexes equally, bottoms aligned; stacks vertically under 760px. See `FormField` for a composed example. Also used to pair an input with its action button:

```jsx
<FieldRow>
  <input placeholder="restyle direction (rewrites every prompt)…" />
  <Button>Restyle</Button>
</FieldRow>
```
