Pill-shaped suggestion button — indigo outline that fills solid on hover. Used for one-tap prompt variations in the shot inspector. Compose inside a `ChipRow`.

```jsx
<ChipRow>
  <Chip title="full prompt shown on hover">golden hour</Chip>
  <Chip>handheld camcorder</Chip>
  <Chip>slow zoom</Chip>
</ChipRow>
```
