The suite's canonical `--suite-*` palette: brand hues with their per-tool assignments (indigo = Synthograsizer, teal = Glitcher/Videorama, fuchsia = Taste Profile), the dark elevation ramp (`--suite-bg` → `--suite-bg-5`), and the three-step text hierarchy. Always color through these variables — never raw hex.
