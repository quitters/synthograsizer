A creative toolkit, authored by hand.

Synthograsizer is an interactive web application for structured prompt engineering and real-time generative art control — inspired by the tactile experience of hardware synthesizers. It features a D-pad navigation model, MIDI and OSC support, batch generation, and embedded p5.js sketches for live visual experimentation.

The suite spans seven tools — a prompt sequencer, glitch studio, multi-agent chat room, browser DAW, metadata manager, and more — all sharing a common JSON template format so ideas can move freely between them.
